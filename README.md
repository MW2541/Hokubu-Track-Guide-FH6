# Hokubu-Track-Guide-FH6
This is an interactive track guide which serves to outline the quickest/ most optimal way to drive the Hokubu Time Attack track in Forza Horizon 6 in my opinion.
It is by no means definitive as factors like the car, driving style and weather will likely impact the way you approach the track. However I have found that the information enclosed in this guide is valid for the majority of situations.
